Tailer workflow¶

This tutorial describes how to use Tailer on High-performance Cluster such as Biowulf or FRCE, then Tailer-analysis locally on RStudio.

Install Tailer¶

You may want to run this into a dedicated environment.

module load python
pip install jla-tailer

Run Tailer¶

Tailer -a annotation.gtf sample.bam

Local Tailer-analysis¶

Download and extract Tailer-analysis

Open in RStudio. Click on Run App.
